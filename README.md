# Arquitecturas Emergentes — Taller Final (Fase A)

Informe en LaTeX. Archivo principal: `main.tex`.

## Cómo compilar (Windows + VS Code)

1. Instalar **MiKTeX**: https://miktex.org/download
2. Instalar la extensión **LaTeX Workshop** (James-Yu) en Visual Studio Code.
3. En la terminal, instalar Perl (necesario para compilar):
   ```powershell
   winget install StrawberryPerl.StrawberryPerl
   ```
4. Reiniciar VS Code, abrir `main.tex` y compilar / ver el PDF con LaTeX Workshop.

Listo. El resultado queda en `main.pdf`.
